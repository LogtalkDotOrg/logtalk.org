Some notes
  • Download numbers are not available for releases older than version 2.8.3.
  • Number of downloads is not the same as number of (potential) users.
  • Users are free to redistribute Logtalk.
  • There is no way of telling downloads from new (potential) users from downloads of users upgrading from previous versions.
  • To make some sense of the statistics, correlate them to the release dates and to the time interval between releases.
  • Logtalk is included with the YAP Prolog compiler (whose download numbers are not included here) and in some CD distributions of Open Source tools.
